# Issue states

We use only a couple of states in the Markdown files, recorded in a `Status:` line near the top of each issue file:

| State         | Meaning                                                            |
|---------------|--------------------------------------------------------------------|
| `ready`       | Fully specified, ready for implementation                          |
| `in-progress` | Working on it, partially implemented, may or may not be committed. |
| `done`        | Finished                                                           |
